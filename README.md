### Explore QUESTION: How are graceful degradation and service workers related?

- Graceful degradation and Service workers are related bc service workers help a web app keep working even when ideal conditions are not available, such as when the user has a slow connection or no internet. Instead of the app completely breaking and crashing, the service worker can use cached files like HTML, CSS, JavaScript, images, and recipe data to still show the page. This is graceful degradation because the app may not have every online feature, but it still provides a usable experience for the user in the event of poor conditions.
